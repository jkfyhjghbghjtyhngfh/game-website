Bugs & Upgrades

Use this file to track known bugs and feature requests for this site.

Format
- Title: Short description
- Type: Bug / Upgrade
- Status: open / in-progress / fixed / wontfix
- Steps to reproduce
- Notes / Links

Example
- Title: Table Tennis embed shows blank
- Type: Bug
- Status: open
- Steps: Open `table-tennis/table-tennis.html` in Chrome; iframe stays blank and shows fallback link
- Notes: The game is hosted on Famobi and may block embedding via framing or CSP. Added a fallback and overlay; needs better UX or hosting assets locally.

How to report
- Use the website page `bugs.html` to submit a new issue; it will open a prefilled GitHub issue for this repository.
- Or create a GitHub issue directly: https://github.com/jkfyhjghbghjtyhngfh/game-website/issues/new
